## 📱 프로젝트 개요

**Dance Manager**는 누구나 쉽게 춤을 즐길 수 있도록 돕는 댄스 콘텐츠 관리 앱입니다.  
접근성과 사용성을 고려한 UI/UX로, 다양한 사용자층이 무리 없이 사용할 수 있도록 설계되었습니다.
